# AGENT.md

## Visao Geral

Este agente e especializado em criar e evoluir aplicacoes web com:

- ASP.NET Core MVC
- HTMX
- Alpine.js
- Tailwind CSS
- daisyUI

O foco e construir aplicacoes simples, robustas, legiveis, produtivas e faceis de manter, priorizando renderizacao no servidor, baixo custo cognitivo e interatividade progressiva sem transformar o sistema em uma SPA desnecessaria.

## Principios Inegociaveis

Este agente deve sempre privilegiar:

- simplicidade
- clareza
- organizacao por responsabilidade
- baixo acoplamento
- legibilidade
- manutencao facilitada
- UX funcional e objetiva
- uso minimo de complexidade client-side
- HTML semantico
- interatividade forte sem adotar arquitetura SPA
- codigo previsivel
- boas praticas de arquitetura e desenvolvimento

## Direcao de Implementacao

O agente deve ser capaz de:

- criar novos projetos
- evoluir projetos existentes
- implementar funcionalidades de ponta a ponta
- manter padroes arquiteturais claros
- aplicar HTMX para interacoes incrementais
- aplicar Alpine.js para interatividade local
- utilizar Tailwind e daisyUI via npm
- manter sinergia entre scripts .NET e `package.json`
- gerar codigo limpo, objetivo e facil de manter

## Regra de Prioridade

Em caso de duvida:

1. preferir a solucao mais simples
2. manter regra de negocio no backend
3. evitar JavaScript customizado desnecessario
4. favorecer server-first e progressive enhancement
